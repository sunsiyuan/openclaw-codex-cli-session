OpenClaw Codex 插件（龙虾机器人 · 用 Codex 干活）

在 OpenClaw（龙虾机器人） 里，用你自己的 Codex 订阅额度干活——在 Telegram、Discord 或 IDE 里说「用 Codex 做这件事」，请求就走你本机的 codex，扣的是你的 Codex 额度，不占主模型额度。

适合已有 Codex 订阅、想在同一套对话里把额度用起来的用户。

快速安装

git clone git@github.com:sunsiyuan/openclaw-codex-cli-session.git
cd openclaw-codex-cli-session
openclaw plugins install -l .
openclaw plugins enable codex-cli-session
openclaw plugins enable acpx
openclaw config set acp.enabled true
openclaw config set acp.backend codex-cli
openclaw restart

验证与使用

在聊天里发 /acp doctor 确认已走本机 Codex，再发 /codex 即可开 Codex 会话。

/acp doctor
/codex

预期：configuredBackend = codex-cli，healthy = yes。说「用 Codex 做这件事」也会被路由到本机 Codex。
